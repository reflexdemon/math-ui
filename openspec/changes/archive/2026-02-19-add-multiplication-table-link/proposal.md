## Why

Users currently have no way to navigate to the multiplication tables feature from the home page. The multiplication tables feature exists but lacks discoverability, limiting its usefulness to users who don't know it exists.

## What Changes

- Add a navigation link from the home page to the multiplication tables feature
- Update home page content to prominently feature the multiplication tables as a key capability
- Improve user flow by making the multiplication tables easily accessible from the landing screen

## Capabilities

### New Capabilities

- None - this is a navigation/content change to existing functionality

### Modified Capabilities

- `multiplication-tables-display`: Adding navigation pathway from home page to this feature

## Impact

- **Frontend**: Home page component modifications
- **Navigation**: New route/link to multiplication tables page
- **User Experience**: Improved discoverability of multiplication tables feature
